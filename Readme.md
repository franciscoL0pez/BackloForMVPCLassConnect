## ⚙️ Instalación y Ejecución del Proyecto

### 1. Instalar `node.js` y `npm` en tu sistema

### 2. Instalar las dependencias
   - Ejecuta el siguiente comando en la carpeta raíz del proyecto:
     ```bash
     npm install
     ```
